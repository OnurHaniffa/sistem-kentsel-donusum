Sıkça Sorulan Sorular

### Sorular ve Cevaplar

**S1: Hangi şehirlerde hizmet veriyorsunuz?**
```
İstanbul ve Kocaeli için yüz yüze + online hizmet veriyoruz. Diğer şehirler için online danışmanlık hizmeti sunuyoruz.
```

**S2: Arsa tapulu olmam engel mi?**
```
Arsa tapulu olması engel değildir. Ancak desteklerden yararlanma durumu farklılık gösterebilir.
```

**S3: Hangi binalar için geçerli?**
```
Riskli tüm yapılar için geçerlidir. Binanızın riskli yapı olarak tespit edilmesi gerekmektedir.
```

**S4: Karar için kaç kişinin onayı gerekiyor?**
```
Salt çoğunluk (%50+1) yeterlidir. Ortaklar kurulu kararı için arsa payı çoğunluğu aranır.
```

**S5: Binamın ruhsatı yok, başvurabilir miyim?**
```
Ruhsat şart değildir. Riskli yapı olarak tespit edilmesi yeterlidir, yeni bina ruhsatlı olacaktır.
```
